# 自动接听电话

### 运行和测试环境
1、Android Studio 3.2  
2、Android 8.1.0  
3、华为畅想9

### 改变地方
1、本程序基于[82367825/CallHepler](https://note.youdao.com/)修改而来  
2、添加了Android 8.1.0 下，动态申请android.permission.ANSWER_PHONE_CALLS
